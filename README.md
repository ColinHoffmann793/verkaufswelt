# 🎮 Verkaufswelt – Erkenne den Kundentyp

Ein kleines **3D-Lernspiel für Technical Sales**: Lauf in der Ego-Perspektive durch einen Park, erkenne den **DISG-Kundentyp** an dem, was die Leute sagen, und antworte typgerecht (Platinregel). Je besser du den Typ triffst, desto mehr verkaufst du – und am 🎡 Glücksrad kannst du dein Glück versuchen.

## ▶️ Jetzt spielen
👉 **https://colinhoffmann793.github.io/verkaufswelt/**

## 🕹️ Steuerung
- **Klick ins Bild** → mit der **Maus** umsehen (**Esc** gibt die Maus frei)
- **WASD** = gehen · **E** = sprechen / Glücksrad drehen
- **Handy:** ziehen zum Umsehen, „Gehen"-Knopf zum Laufen, Schilder antippen

## 🎯 Die vier Kundentypen (DISG)
🔴 **Dominant** · 🟡 **Initiativ** · 🟢 **Stetig** · 🔵 **Gewissenhaft**

Pro Aussage gibt es **eine voll passende** und **eine halb passende** Antwort – erkenne sie aus der Sprache des Kunden.

## ⚙️ Technik
Alles steckt in einer einzigen `index.html` (Three.js eingebettet) – läuft komplett im Browser, **kein Server nötig**. Inhaltlich abgeleitet aus dem Schulungsskript „Technical Sales V8.0".
